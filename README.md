# Code Sandbox (WPF)

Simple WPF application for executing code via HackerEarth API.

## Features
- WPF + MVVM
- Async code execution
- Error handling
- Clean architecture

## API Key
⚠️ HackerEarth API key is not included.
Provide your own API key locally in `HackerEarthService`.

## Tech Stack
- C#
- .NET
- WPF
